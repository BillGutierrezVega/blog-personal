# Mi Proyecto Django 🚀

Este es un proyecto web desarrollado con Django que incluye funcionalidades como la creación de publicaciones, etiquetas y autenticación de usuarios.

## Funcionalidades 🛠️

- 📝 Crear, editar y eliminar publicaciones.
- 🔖 Asociar etiquetas a las publicaciones.
- 🏷️ Crear y gestionar etiquetas.
- 👤 Registro e inicio de sesión de usuarios.

## Requisitos 📋

- Python 3.x
- Django 3.x

## Instalación 🚨

1. 🧙 Clona el repositorio:

   ```bash
   git clone https://github.com/tu-usuario/mi-proyecto-django.git
  2. **Activa el entorno virtual:**

   - En Windows:

     ```bash
     .\venv\Scripts\activate
     ```

   - En Linux/Mac:

     ```bash
     source venv/bin/activate
     ```

3. **Instala las dependencias:**

    ```bash
    pip install -r requirements.txt
    ```

4. **Aplica las migraciones:**

    ```bash
    python manage.py migrate
    ```

5. **Inicia el servidor de desarrollo:**

    ```bash
    python manage.py runserver
    ```

6. **Abre tu navegador y visita [http://localhost:8000/](http://localhost:8000/).**
